## seek — frontend

#### React + TypeScript frontend for seek, a RAG (Retrieval-Augmented Generation) application that lets users upload documents and ask natural-language questions about their content.

#### Overview

seek is a client-based application that allows you to:
1. Upload a document (PDF, DOCX, PPTX, TXT).
2. Select the document to be queried, from the sidebar.
3. Ask natural-language questions about its content in a chat interface.
4. Receive answers generated by an AI model (Gemini), grounded in the document's actual content (via the backend's RAG pipeline).

#### Project Structure

src/
 ├── api/
 │    └── documentAPI.ts      — all fetch() calls to the backend REST API
 ├── components/
 │    ├── Upload.tsx          — sidebar: upload button, document list, delete
 │    └── Chat.tsx            — main panel: message history, question input
 ├── types.ts                 — shared TypeScript interfaces
 ├── App.tsx                  — top-level state and layout
 └── App.css                  — styling

#### Features

1. Upload PDF, DOCX, PPTX, or TXT files.
2. View a list of previously uploaded documents.
3. Select a document to start a chat session about it.
4. Ask questions and receive AI-generated, context-grounded answers.
5. Delete documents.
